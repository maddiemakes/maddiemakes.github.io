Tuples

A tuple is a finite ordered list of elements.

A k-tuple is an ordered sequence (a1, a2,...,ak) of k elements. The order matters and repitition matters so that (1,2,3) is not the same tuple as (3,2,1).

Ordered pairs are 2-tuples and ordered triples are 3-tuples.

First Order Logic

Predicates can have variables as arguments. Predicate P has argument x in P(x).

Operations

Quantifier Meaning
∀x for all x
∃x there exists an x
∄x there does not exist an x

Examples

Formula Translation
P(x) It is raining on day x
Q(x) The sun does not shine on day x
P(x)→Q(x) If it is raining on day x then the sun does not shine on day x
∀x(P(x)→Q(x)) For all x if it is raining on day x then the sun does not shine on day x
P(x)∧Q(x) It is raining on day x and the sun is not shining on day x
∃x(P(x)∧Q(x)) There is a day on which it is raining on day x and the sun does not shine on day x

Semantics

An interpretation of a first-order formula has the following parts:

  • Quantification domain which gives the values that can be assigned to the variables
  • An assignment of a meaning to each predicate, function, and constant symbol:
    • If I is an interpretation, then PI is the meaning of predicate P in I. This assigns a value of true or false to each combination of arguments as elements of the domain.
    • If f is a function symbol, then fI is the meaning of the function f in I. This assigns an element of the domain to each combination of the arguments as elements of the domain.

Validity: Hello.

  • A formula is valid if it is satsified by all interpretations
  • A formula is satisfiable if it is satisfied by at least one interpretation
  • Two formulas are equivalent if they are satisfied by the same interpretations, or worlds
