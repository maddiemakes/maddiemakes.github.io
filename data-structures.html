Linked List

A linked list is like a backwards array list. The LinkedList knows firstLink, which is the most-recently inserted Link.

Insert

To add a new Link to the LinkedList, we just set the new Link's next to the current firstLink and then set firstLink to the new Link.

Remove

To remove a Link from a LinkedList, we set the next of the previous Link (the one before the Link being removed) to the next of the Link being removed. If the Link being removed is firstLink, then just set firstLink to firstLink.next.

Time Complexities

Algorithm Average Worst Case
Insert O(N) O(N)
Remove O(N) O(N)
Find O(N) O(N)

Binary Search Tree (BST)

A binary search tree is a way of storing data in a tree formation. Each piece of data in the BST is referred to as a node and the node at the top of the tree is referred to as root. Each node has children, the nodes below it, and every child has a parent, the node above it. The child to the left is referred to as the left child, and the child to the right is the right child. If a node has no children, it is referred to as a leaf node.

Insert

When you insert a new node into a BST, you compare its value to each node, starting at the root. If the new node is less than the comparing node, move to the left. If the new node is greater than the comparing node, move to the right. If the comparing node does not have any children, insert the new node here.

Remove

When you remove a node from a BST, there are several cases.

  • If the node being removed is a leaf node, just remove it.
  • If the node being deleted has only one child, then replace the node with said child.
  • If the node being deleted has two children, replace it with the left child.

Formulas
Max height N-1
Min height floor(log N)

AVL Tree

Balanced BST

When you insert, sometimes the tree becomes imbalanced. In order to balance it, count up from the inserted node and find the first imbalanced one (usually two parents up). From there, count the movements to the inserted node (either left or right). Refer to the table below for how to rotate.

Node movement Rotation solution Node(s) to rotate
LL R Middle
RR L Middle
LR LR Bottom then middle
RL RL Bottom then middle

Time Complexities

Algorithm Average Worst Case
Insert O(log N) O(log N)
Remove O(log N) O(log N)
Find O(log N) O(log N)

Splay Tree

Nearly balanced BST

In a splay tree, whenever a node is accessed, the tree is "splayed," or reorganized so that the accessed node becomes root. This allows for faster access of recently accessed nodes.

Note: in find(), if the node is not found, still splay the last node accessed.

Time Complexities

Algorithm Average Worst Case
Insert O(log N) O(N)
Remove O(log N) O(N)
Find O(log N) O(N)
Splay O(log N) O(N)
